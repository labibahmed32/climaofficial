 Installation Guide


    

  1. Add the Landing Page Script in your page <head> — it auto-detects which package button is clicked and stores the variant
    2. 

  2. For BuyGoods: Paste Checkout Script in Dashboard → Funnel Pixels → Checkout (captures name, email, phone, address, card data)
    3. 

  3. Add each Upsell Script on its respective upsell page — Upsell 1 automatically creates the sale record, rest track accept/decline
    4. 

  4. Add Thank You Page Script on the BuyGoods thank-you page (last page after all upsells) — confirms the buyer completed the full funnel
    5. 

  5. Use the Test Tracking section below to verify everything works
    6.
