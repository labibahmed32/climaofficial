Postback URL Documentation

How it works: When a conversion occurs, the advertiser fires a GET request to your postback URL with the click_id. This page looks up the click, finds the associated offer and affiliate, and creates a conversion record automatically.

Postback URL Format

https://YOUR-DOMAIN/api/postback?click_id={click_id}&secret=YOUR_SECRET&payout={payout}&txn_id={txn_id}

Parameters

ParameterRequiredDescription
click_idYesClick ID from the tracking link (passed to advertiser via {click_id} macro)
secretYesSecret key (set in Admin > Settings > Postback)
payoutNoCustom payout amount (uses offer default if not set)
revenueNoCustom revenue amount (uses offer default if not set)
txn_idNoExternal transaction ID from the advertiser
statusNoapproved (default) or pending

Example

/api/postback?click_id=c1a2b3c4d5e6&secret=mysecretkey&payout=25.00&txn_id=TXN-12345
Note: This page processes postbacks client-side. For true server-to-server (S2S) postbacks, use a Cloudflare Worker or Cloud Function. Works for testing and browser-based calls.
